# Instagram Login Page Clone

An Android application that replicates the Instagram login page UI using LinearLayout only.

## Features
- Clean Instagram-style UI
- Username/Email/Phone input field
- Password input field
- Login button with Toast message
- Forgot password option
- Facebook login option
- Sign up link

## Technologies Used
- Android Studio
- Kotlin
- XML (LinearLayout only)

## How to Run
1. Clone this repository
2. Open in Android Studio
3. Run on emulator or physical device

## Code Files
- `MainActivity.kt` - Contains login logic and Toast functionality
- `activity_main.xml` - UI layout using LinearLayout
